# XTC_box
小天才专用工具箱

本项目由魔笑鬼本人制作(不存在开发团队)

在使用之前您可能需要安装pyqtfluentui库(一个好用的ui模板)

使用以下代码安装:
```shell
pip install "PyQt-Fluent-Widgets[full]" -i https://pypi.org/simple/
```

你也可以访问https://pyqt-fluent-widgets.readthedocs.io/zh_CN/latest 获取更多教程
